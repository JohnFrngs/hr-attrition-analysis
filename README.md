# HR Attrition Analysis 👥

## Project Overview
This project focuses on identifying the key drivers behind employee turnover (attrition). Using data-driven insights, we analyze various factors such as job satisfaction, overtime, salary, and years at the company to understand why employees leave and how a company can improve retention.

## Key Insights
* **Overtime:** Employees working significant overtime hours show a higher probability of attrition.
* **Monthly Income:** A clear correlation exists between lower income brackets and increased turnover rates.
* **Work-Life Balance:** Specific job roles with poor work-life balance scores are identified as high-risk areas.

## Tech Stack
* **Language:** Python
* **Environment:** VS Code (Jupyter Notebooks)
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
* **Export Format:** HTML for web-based interactive viewing

## Repository Structure
* `HR_Attrition_Analysis.html`: An exported, easy-to-read version of the analysis including all visualizations.
